# Dossier inicial

Este directorio contiene documentación y guías técnicas del proyecto HexaPlay. A continuación se listan los documentos existentes y su propósito:

- **00_visio_general.md**: describe la visión general del proyecto, los componentes de hardware (Raspberry Pi 3, pantalla, altavoces, micrófono, dongle Bluetooth, microSD, fuente de alimentación, botonera) y las piezas de software (Crankshaft/OpenAuto, BlueZ, PulseAudio, Xorg, Qt/QML, bash y Python, systemd). Incluye un diagrama conceptual y glosario.
- **05_repo_basics.md** / **05_git_basics.md**: guía de uso básico del repositorio y de Git. Explica qué es un repositorio Git, cómo clonar este repositorio, el flujo de trabajo básico (pull, branch, add, commit, push, merge) y cómo editar/añadir archivos desde la web de GitHub. Además incluye sugerencias sobre la estructura de ramas y buenas prácticas.
- *Próximos documentos*: se añadirán archivos detallados sobre el historial de decisiones, la instalación del sistema, la guía de audio Bluetooth, el branding de la interfaz, los servicios de arranque, problemas y soluciones (troubleshooting) y las tareas pendientes.

Si buscas la estructura completa del proyecto, consulta el documento principal en la conversación de ChatGPT o revisa el árbol de carpetas en el repositorio.
